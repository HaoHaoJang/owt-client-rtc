# OWT Client [直接下载](#下载)

多人互动即时通讯Web版 和 Electron跨平台版客户端，服务器程序使用的OWT-Server [Open WebRTC Toolkit](https://github.com/open-webrtc-toolkit/)

## 特性

- PC多平台支持（Windows、Linux、MacOS）
- 视频聊天
- 文字聊天
- 多摄像头支持
- 桌面分享
- 本地录制
